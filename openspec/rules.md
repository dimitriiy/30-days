# Global Rules for AI Agents

- Не изменять поведение без явной просьбы: при рефакторинге сохранять публичные API‑контракты.
- Не добавлять новые внешние зависимости без обновления спецификаций и моего явного подтверждения.
- Для всех изменений в `/app` соблюдать текущую структуру маршрутов и соглашения по layout.
- Всегда сначала читать `openspec/project.md` и актуальный `openspec/changes/.../proposal.md` перед тем, как менять код.
